## مثال 2: إنشاء وصف لملابس

### البيانات المرسلة
```json
{
  "productName": "تي شيرت قطن عالي الجودة",
  "category": "Fashion",
  "price": 150,
  "tags": ["تي شيرت", "قطن", "رجالي", "كاجوال"]
}
```

### الوصف المولد
```
تي شيرت فخم مصنوع من قطن 100% عالي الجودة، يتمتع بنعومة فائقة والراحة القصوى. التصميم البسيط والأنيق يناسب جميع المناسبات الكاجوال. يتميز بخياطة قوية وألوان ثابتة لا تتلاشى بسهولة. مناسب للرجال من جميع الأعمار ويعطي مظهراً عصرياً وراقياً. يمكن ارتداؤه مع البنطلونات أو الشورتات للحصول على إطلالة مريحة وأنيقة في نفس الوقت.
```

## مثال 3: إنشاء وصف لمنتج غذائي

### البيانات المرسلة
```json
{
  "productName": "عسل نحل طبيعي 100%",
  "category": "Food & Beverages",
  "price": 200,
  "tags": ["عسل", "طبيعي", "صحي", "عضوي"]
}
```

### الوصف المولد
```
عسل نحل نقي وطبيعي 100% من أجود الأنواع، منتج بدون إضافات أو معالجات كيميائية. مستخرج مباشرة من خلايا النحل الصحية ويتمتع بفوائد صحية عديدة. غني بالمعادن والفيتامينات الطبيعية والمضادات الحيوية. يستخدم في العسل الطبيعي للتحلية الصحية وتقوية المناعة وعلاج الكحة والحلق. طعمه ناعم ورائحته عطرة تميزه. مناسب للأطفال والكبار ولا يسبب أي آثار جانبية.
```

## مثال 4: خطأ - المنتج غير موجود

```

  "message": "Product not found",
  "statusCode": 404
}
```

## تدفق العملية (User Flow)

```
┌─────────────────────────────────────────────────────────────┐
│ 1. Admin يدخل لتحرير المنتج                                  │
│    - يرى حقل الوصف مع زر "إنشاء بـ AI" ✨                    │
└────────────────┬────────────────────────────────────────────┘
                 │
┌────────────────▼────────────────────────────────────────────┐
│ 2. يضغط على زر "إنشاء بـ AI"                                 │
│    - الزر يصبح معطلاً                                        │
│    - يعرض "جاري الإنشاء..."                                 │
│    - يظهر spinner أثناء الانتظار                             │
└────────────────┬────────────────────────────────────────────┘
                 │
┌────────────────▼────────────────────────────────────────────┐
│ 3. يتم إرسال الطلب للـ Backend                               │
│    POST /api/products/:id/generate-description              │
│    - البيانات: الاسم، الفئة، السعر، الوسوم                  │
└────────────────┬────────────────────────────────────────────┘
                 │
┌────────────────▼────────────────────────────────────────────┐
│ 4. Backend يستدعي خدمة Python AI                            │
│    POST http://localhost:8000/api/generate-description      │
│    - الانتظار 3-10 ثواني                                     │
└────────────────┬────────────────────────────────────────────┘
                 │
┌────────────────▼────────────────────────────────────────────┐
│ 5. AI تولد الوصف                                            │
│    - تستخدم نموذج Llama 3 أو Mistral                        │
│    - ترجع وصف احترافي                                      │
└────────────────┬────────────────────────────────────────────┘
                 │
┌────────────────▼────────────────────────────────────────────┐
│ 6. Backend يحدث المنتج                                      │
│    - يحفظ الوصف الجديد                                      │
│    - يضع aiGenerated.description = true                     │
└────────────────┬────────────────────────────────────────────┘
                 │
┌────────────────▼────────────────────────────────────────────┐
│ 7. Dashboard يستقبل الاستجابة                               │
│    - يحدث حقل الوصف بالنص الجديد                            │
│    - يعرض إشعار نجاح ✅                                      │
│    - الزر يعود للحالة العادية                               │
└────────────────┬────────────────────────────────────────────┘
                 │
┌────────────────▼────────────────────────────────────────────┐
│ 8. Admin يرى الوصف المولد                                   │
│    - يمكن تعديله إذا لزم الأمر                              │
│    - يضغط Save لحفظ المنتج                                 │
└─────────────────────────────────────────────────────────────┘
```

## الرسائل الظاهرة للمستخدم

### ✅ نجاح
```
✨ تم إنشاء الوصف باستخدام الذكاء الاصطناعي
```

### ❌ خطأ
```
❌ فشل في توليد الوصف
Failed to connect to AI service
```

### ⚠️ تحذير
```
يرجى إدخال اسم المنتج أولاً
```

---

## حالات الاستخدام

### ✅ الحالات المدعومة:
- توليد الوصف لمنتج جديد
- تحديث الوصف لمنتج موجود
- إعادة توليد الوصف عدة مرات
- دعم منتجات من جميع الفئات

### ❌ الحالات غير المدعومة (حالياً):
- توليد وصف قبل حفظ المنتج
- توليد وصف لمنتجات محذوفة
- توليد وصف بدون بيانات المنتج

---

## إحصائيات الاستخدام

سيتم تسجيل:
- عدد مرات توليد الوصف
- الفئات التي تم توليد وصف لها
- متوسط طول الوصف المولد
- معدل القبول (هل يتم تعديل الوصف بعد التوليد)

---

## نصائح للحصول على وصف أفضل

1. **اسم واضح**: استخدم اسم واضح ووصفي للمنتج
2. **فئة صحيحة**: اختر الفئة الصحيحة للمنتج
3. **وسوم مفيدة**: أضف وسوم وصفية مثل اللون والحجم
4. **السعر معقول**: السعر يساعد على تقدير جودة الوصف
